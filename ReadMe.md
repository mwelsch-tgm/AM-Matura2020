# Overleaf Backup Solution

## Requirements
* Chrome installed
* Compatible [ChromeDriver](https://chromedriver.chromium.org/downloads) installed
* JSON-File `test.json` with following structure:
```json
{
  "email": "<your@mail>",
  "password": "<password>",
  "document": "<document_title>"
}
```
